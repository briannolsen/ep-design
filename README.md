# EP Design

Landing page para EP Design, una agencia de gestion de redes sociales y marketing digital. Presenta un diseno oscuro y moderno con animaciones de scroll fluidas impulsadas por GSAP.

## Stack Tecnologico

- HTML5
- CSS3
- JavaScript Vanilla
- GSAP + ScrollTrigger

## Caracteristicas

- Efectos de cursor personalizado
- Animaciones activadas por scroll
- Botones magneticos
- Contadores animados
- Carrusel de logotipos de clientes
- Diseno responsive con menu movil
- Revelado palabra por palabra del texto hero

## Secciones

- Hero
- Problema
- Solucion
- Servicios
- Resultados
- Logotipos de Clientes
- Testimonios
- Portafolio
- CTA

## Como Ejecutar

Abrir `index.html` directamente en un navegador o utilizar un servidor local:

```bash
npx serve .
```
